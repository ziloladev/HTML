Introduction

HTML is an acronym for HyperText Markup Language, and it is the standard language for creating web pages. Every web page you surf on your browser is built using HTML. The primary use of HTML is to provide a proper structure to the web page so all the content or data of the page could represent adequately. A stand-alone HTML can create only static and skeleton-looking black and white pages, but with the help of CSS and JavaScript, we can create a more interactive and intuitive web page. When we try to visit a website or click on the link, we basically request the server to show us the page, then the server acts on our request and sends us an appropriate HTML document as a response. Then this HTML document is parsed by the browser, and we are able to see the content.

HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph</p>

</body>
</html>

HTML Elements

The HTML elements provide semantic meaning to the webpage content. We often use the terms 'HTML elements' and 'tags' interchangeably, but technically, they are different. An HTML tag is just a character inside angle brackets <>, whereas an HTML element consists of a starting tag, its attributes, content, and an end tag. For example:

<p class= "para"> Hello World <p>

HTML Attributes

In HTML, attributes are used to provide additional information about elements. For most HTML elements, attributes are optional, but some elements require them. Attributes are always specified within the opening tag of an HTML element and are defined as a name-value pair. Example

<image src="cat.jpg" alt = "cat image">

"In this example, src="cat.jpg" and alt="cat image" are two attributes where src and alt are attribute names, and "cat.jpg" and "cat image" are attribute values. Here, the alt attribute is optional, but src is mandatory because it specifies which image to show. HTML Heading

"To display a section heading, title, or subtitle, we can use HTML heading tags. In HTML5, there are six heading tags, ranging from <h1> to <h6>, where <h1> specifies the largest heading and <h6> represents the smallest or subheading. If the content is enclosed within heading tags, it will be displayed larger and bolder compared to other text on the webpage. Example:"

<h1>First Heading </h1>
<h2>Second Heading </h2>
<h3>Third Heading </h3>
<h4>Forth Heading </h4>
<h5>Fifth Heading </h5>
<h6>Sixth Heading </h6>

HTML Paragragh

In HTML paragraphs can be defined using <p> element. Paragraph text always starts from a new line, the browser parsed the <p> tag and automatically add some margin and white space after the end </p> tag.

<p> Hello! and Welcome to TechGeekBuzz </p>
<p> Here you get to know all about the latest technology. </p>

HTML Formating

In HTML, we have many special elements that can provide special meaning to text content.

HTML Elements Description
<b> Bold the text
<strong> An alternative for <b>, which tell that this text is essential.
<i> Italic the text
<em> Similar to italic but used when we want to emphasize the text.
<mark> It marks the text with a default "yellow" background colour
<small> Decrease the text size
<del> It prints a cross line over the text.
<ins> Represents the inserted text by putting an underline.
<sub> This element is used to display the subscript.
<sup> It can make a text superscript.
